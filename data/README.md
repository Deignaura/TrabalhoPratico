# Dados

Esta pasta guarda arquivos de persistência simples em texto.

## Arquivos

| Arquivo | Conteúdo |
|---|---|
| `recorde.txt` | Melhor distância registrada em metros |
| `ranking.txt` | Top 5 jogadores por distância percorrida |
| `ranking_moedas.txt` | Top 5 jogadores por moedas coletadas em uma partida |
| `moedas.txt` | Total de moedas acumuladas pelo jogador (usadas na loja) |

## Formato dos rankings

Cada linha segue o padrão `nome,valor`, ordenado do maior para o menor valor.

## Observação

Evite versionar dados pessoais reais dos jogadores.
